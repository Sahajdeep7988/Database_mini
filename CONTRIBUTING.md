# Contributing to RelictDB

This is a personal/educational project. Contributions, suggestions, 
and issue reports are welcome.

## Reporting Issues
- Use GitHub Issues for bug reports
- Include your OS, compiler version, and the exact command that failed

## Suggesting Features
Open an issue with the tag [FEATURE] and describe:
1. What you want to add
2. Why it would be useful
3. A rough idea of how it could be implemented

## Code Style
- C++17 standard
- Snake_case for variables and functions
- PascalCase for class names
- All public methods must have a comment explaining their purpose
